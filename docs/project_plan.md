# ABCD サポート担当コパイロット PoC プロジェクト計画書

## プロジェクト概要

### 目的
ABCD (Action-Based Conversations Dataset) を活用し、対話履歴から「次に取るべきアクション」を推論するサポート担当コパイロットの技術検証を実施する。

### 開発期間
- **期間**: 2025年7月27日 ～ 2025年8月11日 (16日間・2週間)
- **工数見積**: 約80人時（1日5時間想定）

### 成功基準
1. **技術的成功基準**
   - ABCDデータセットの正常取り込み・前処理完了
   - Action State Tracking精度50%以上達成
   - エンドツーエンドでの推論パイプライン構築

2. **ビジネス的成功基準**
   - 対話→アクション推論の実現可能性確認
   - 本格開発に向けた技術的課題の明確化
   - コスト・工数見積もりの精緻化

## リスクと対策

### 高リスク項目

#### 1. データ品質・可用性リスク
- **リスク**: ABCDデータの品質問題、取得困難
- **対策**: 
  - 早期段階でのデータ確認・分析
  - 代替データソースの事前調査
- **影響度**: 高（プロジェクト全体に影響）

#### 2. 精度目標未達リスク
- **リスク**: Action State Tracking精度50%未達
- **対策**: 
  - 複数アプローチの並行検証
  - 段階的な精度改善
- **影響度**: 中（要件調整で対応可能）

#### 3. 技術的課題リスク
- **リスク**: SageMaker・Snowflake連携の技術的困難
- **対策**: 
  - 早期のPoC・検証
  - エキスパートへの相談体制
- **影響度**: 中（アーキテクチャ調整で対応）

### 中リスク項目

#### 4. 工数オーバーリスク
- **リスク**: 想定より複雑で工数超過
- **対策**: 
  - 週次進捗レビュー
  - 優先度に基づく機能調整
- **影響度**: 中

#### 5. 環境・権限リスク
- **リスク**: AWS・Snowflakeアクセス権限の遅延
- **対策**: 
  - プロジェクト開始前の権限申請
  - 代替環境の準備
- **影響度**: 低（開始遅延のみ）
